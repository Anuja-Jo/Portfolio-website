# 🌐 Personal Portfolio Website

This is my personal portfolio website built using the MERN stack. It showcases my projects, skills, and contact information.

---

## 🚀 Features

- 👤 About Me section
- 🛠️ Skills section
- 📂 Projects showcase
- 📧 Contact form
- 📱 Responsive design

---

## 🛠️ Tech Stack

- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB

---

## ⚙️ Setup Instructions

```bash
git clone https://github.com/Anuja-Jo/Portfolio.git

cd client
npm install
npm run dev

cd ../server
npm install
npm start
